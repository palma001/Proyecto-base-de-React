# Q tienda - app

Aplicación web para la tienda de QTiendas, desarrollada con React, por [Qbits](https://github.com/Qbits).

## Clonar repositorio

```bash
git clone https://github.com/Qbits/q_tienda.git
```

## Instalar dependencias

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Compilación | opcional

```bash
npm run build
```
